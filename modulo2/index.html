Árvore Soluções de Software

Para a programação, Engenharia de Software e Ciências da Computação, árvore está contextualizada como uma das mais importantes estruturas de dados não lineares

Uma árvore é formada por um conjunto de elementos que armazenam informações chamados nodos ou nós. Toda a árvore possui o elemento chamado raiz, que possui ligações para outros elementos denominados ramos ou filhos. Estes ramos podem estar ligados a outros elementos que também podem possuir outros ramos. O elemento que não possui ramos é conhecido como nó folha, nó terminal ou nó externo. O que justifica os motivos de ser usado a árvore como metáfora.
